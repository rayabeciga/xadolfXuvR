# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行检测
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持长度测量功能
- 支持选择检测模型
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 通过标定测量固定距离下平视目标的长度
- 提供完整源码、UI界面源文件和项目运行教程

## 适用场景
- 需要进行目标检测和长度测量的工业检测场景
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 适用于需要实时显示目标位置、目标总数、置信度等信息的应用场景

## 技术特点
- 基于YOLOv8模型实现
- 采用Python 3.9开发环境
- 依赖opencv-python、PyQt5、ultralytics、torch1.9等库
- 提供图形用户界面，操作简便
- 已训练好模型，配置成功环境可直接使用
